# Vue Calendar Component

Компонент календаря для Vue.js (версия 2 и выше) с возможностью переключения месяцев, выбора даты и смены языка интерфейса.

## Возможности

- ✅ Переключение месяцев (предыдущий/следующий)
- ✅ Выбор конкретной даты при клике
- ✅ Событие выбора даты с возвратом значения в формате "год-месяц-день"
- ✅ Инициализация с начальной датой
- ✅ Поддержка нескольких языков (русский, английский, немецкий, французский)
- ✅ Современный и адаптивный дизайн
- ✅ Подсветка текущего дня
- ✅ Подсветка выбранной даты

## Использование

### Базовый пример

```vue
<template>
  <div>
    <VueCalendar @date-selected="onDateSelected" />
  </div>
</template>

<script>
import VueCalendar from './components/VueCalendar.vue'

export default {
  components: {
    VueCalendar
  },
  methods: {
    onDateSelected(date) {
      console.log('Выбранная дата:', date) // Формат: "2024-12-25"
    }
  }
}
</script>
```

### С начальной датой

```vue
<template>
  <VueCalendar 
    :initial-date="'2024-12-25'" 
    @date-selected="onDateSelected" 
  />
</template>
```

### С выбором языка

```vue
<template>
  <VueCalendar 
    :locale="'en'"
    @date-selected="onDateSelected" 
  />
</template>
```

### Комбинированный пример

```vue
<template>
  <VueCalendar 
    :initial-date="'2024-06-15'" 
    :locale="'de'"
    @date-selected="onDateSelected" 
  />
</template>
```

## Props (Свойства компонента)

| Свойство | Тип | По умолчанию | Описание |
|----------|-----|--------------|-----------|
| `initial-date` | String | `null` | Начальная дата в формате "год-месяц-день" (например, "2024-12-25"). Если не указана, используется текущая дата |
| `locale` | String | `'ru'` | Язык интерфейса. Доступные значения: `'ru'`, `'en'`, `'de'`, `'fr'` |

## Events (События)

| Событие | Описание | Возвращаемое значение |
|---------|----------|----------------------|
| `date-selected` | Срабатывает при клике на день | Строка в формате "год-месяц-день" (например, "2024-12-25") |
| `dateSelected` | Альтернативное название события (camelCase) | То же самое |

## Поддерживаемые языки

- `ru` - Русский
- `en` - English (Английский)
- `de` - Deutsch (Немецкий)
- `fr` - Français (Французский)

## Особенности реализации

1. **Инициализация даты**: Компонент автоматически парсит дату в формате "год-месяц-день" и устанавливает текущий месяц и день. Если дата не передана, используется текущая дата.

2. **Переключение месяцев**: Кнопки со стрелками позволяют переключаться между месяцами. При выборе даты из другого месяца календарь автоматически переключается на нужный месяц.

3. **Выбор даты**: При клике на день компонент эмитит событие `date-selected` с выбранной датой в формате "год-месяц-день".

4. **Многоязычность**: Компонент поддерживает несколько языков. Названия месяцев и дней недели автоматически меняются в зависимости от выбранного языка.

5. **Визуальная обратная связь**: 
   - Текущий день выделяется синим цветом
   - Выбранная дата выделяется темно-синим цветом
   - Дни из других месяцев отображаются серым цветом

## Кастомизация стилей

Компонент использует scoped стили, но вы можете переопределить их, используя CSS переменные или добавив свои стили. Основные классы для кастомизации:

- `.vue-calendar` - основной контейнер
- `.calendar-header` - заголовок с навигацией
- `.calendar-day` - день календаря
- `.calendar-day.today` - текущий день
- `.calendar-day.selected` - выбранный день
- `.calendar-day.other-month` - день из другого месяца
