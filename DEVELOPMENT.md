# Vue Calendar Component - Документация по шагам разработки

## Шаг 1: Инициализация проекта

Создан `package.json` с необходимыми зависимостями:
- Vue.js версии 2.6.14
- Vue CLI Service для разработки
- Vite для быстрой разработки

## Шаг 2: Создание компонента календаря

Создан компонент `VueCalendar.vue` в директории `src/components/` со следующими возможностями:

### 2.1. Структура данных
- `currentDate` - текущая отображаемая дата (для определения месяца и года)
- `selectedDate` - выбранная пользователем дата
- `locales` - объект с переводами для разных языков

### 2.2. Вычисляемые свойства (computed)
- `currentYear` - текущий год
- `currentMonth` - текущий месяц (0-11)
- `weekdays` - массив названий дней недели в зависимости от языка
- `calendarDays` - массив дней для отображения в календаре:
  - Включает дни предыдущего месяца (для заполнения первой недели)
  - Дни текущего месяца
  - Дни следующего месяца (для заполнения последней недели)
  - Отмечает текущий день и выбранную дату

### 2.3. Методы навигации
- `previousMonth()` - переход к предыдущему месяцу
- `nextMonth()` - переход к следующему месяцу

### 2.4. Метод выбора даты
- `selectDate(date)` - обработка клика по дню:
  - Сохраняет выбранную дату
  - Переключает месяц, если выбранная дата из другого месяца
  - Эмитит событие `date-selected` с датой в формате "год-месяц-день"

### 2.5. Инициализация
- `initializeDate()` - инициализация компонента:
  - Парсит `initialDate` из пропса (формат "год-месяц-день")
  - Если дата не передана, использует текущую дату
  - Устанавливает выбранную дату

### 2.6. Поддержка языков
- Метод `getMonthName()` - возвращает название месяца в зависимости от языка
- Объект `locales` содержит переводы для:
  - Русского языка (ru)
  - Английского языка (en)
  - Немецкого языка (de)
  - Французского языка (fr)

## Шаг 3: Стилизация

## Использование компонента

### Минимальный пример:
```vue
<VueCalendar @date-selected="handleDateSelection" />
```

### С параметрами:
```vue
<VueCalendar 
  :initial-date="'2024-12-25'"
  :locale="'en'"
  @date-selected="handleDateSelection" 
/>
```

### Обработка события:
```javascript
methods: {
  handleDateSelection(date) {
    // date в формате "2024-12-25"
    console.log('Выбрана дата:', date)
  }
}
```

## Запуск проекта

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер разработки:
```bash
npm run dev
```

3. Откройте браузер по адресу http://localhost:3000

## Тестирование функциональности

1. **Переключение месяцев**: Нажмите кнопки "<" и ">" в заголовке календаря
2. **Выбор даты**: Кликните на любой день календаря
3. **Смена языка**: Измените проп `locale` на другой язык (ru, en, de, fr)
4. **Инициализация с датой**: Передайте проп `initial-date` с датой в формате "год-месяц-день"

## Возможности для расширения

Компонент можно расширить следующими функциями:
- Выбор диапазона дат
- Блокировка определенных дат

- Кастомизация цветов через props
- Добавление дополнительных языков
